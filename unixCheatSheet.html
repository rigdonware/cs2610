Unix Cheat Sheet

By: Rigdon Ware

This site is used for beginners to learn basic commands on a Unix terminal

Command Definition Example
mkdir [filename] Creates a new Directory 
$mkdir NewFolder
$ls
NewFolder unixCheatSheet.html hello-world.html README.txt
rm -rf [filename] Removes a file 
$rm -rf hello-world.html
$ls
NewFolder unixCheatSheet.html README.txt
ls Shows what is inside the folder you are currently in 
$ls
NewFolder unixCheatSheet.html README.txt
pwd Prints the working directory 
$pwd
~/workspace/cs2610/UnixCheatSheet
cd [filepath] Allows you to choose a new directory 
$cd NewFolder
$pwd
~/workspace/cs2610/UnixCheatSheet/NewFolder
